## Validation Tips

1. **Search Strategy**:
   - Look for christening records first (more common than birth records pre-1837)
   - Try variants: "Jeffery" vs "Jeffrey", "John" vs "Jonathan"
   - Broaden location if exact parish yields nothing

2. **What to Extract**:
   - Full name as shown in record
   - Event date (christening/birth)
   - Parish/location
   - Father's full name
   - Mother's full name (including maiden name if shown)
   - Record source citation

3. **Validation Criteria**:
   - ✅ **Perfect Match**: All fields match exactly
   - ⚠️ **Partial Match**: Name + location match, date within 2 years
   - ❌ **No Match**: No suitable record found

4. **Common Issues**:
   - **Name Variations**: Different spellings were common
   - **Date Ranges**: Birth date might be estimated, christening actual
   - **Missing Records**: Some parishes have gaps in records
   - **Multiple Matches**: Choose closest date/location match
